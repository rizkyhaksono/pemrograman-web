## Getting Started

### Prerequisites

- HTML 5
- CSS 3
- Javascript
- PHP
- Laravel
- Composer
- ReactJS
- IDE (VSCode, etc.)

### Installing

1. Clone this repository
2. Open the project in your IDE
3. Run the project
4. Choose the module you want to run
